    Андроид приложения

    YouTube Vanced - Приложение позволяющее без рекламы смотреть youtube, так же умеет сам проматывать рекламные интеграции в видео. Работает функция фонового воспроизведения и воспроизведения в окне. Так же есть аналог для youtube music, устанавливать лучше всего через Vanced manager. Советуем отключить автообновление этих приложений в маркете, иначе придёться устанавливать vanced каждый раз после обновления.

    Newpipe - Клиент youtube, открытый, без рекламы, с фоновым просмотром, загрузкой видео и неудобными подсказками видео.

    Termux - эмулятор терминала с пакетным менеджером, можно даже поставить линукс через Anlinux

    X-plore File Manager - Ультрагодный файловый менеджер, архиватер, ходитель по сетевым дискам, раздаватель по wifi и пр. примиком с симбы. Бесплатная версия не особо и ограниченна, но есть взломанные.

    Aurora Store - Кливент гугл стора с доступом к нему без аккаунта. Если не работает проверь, что установлена последняя версия.

    Aurora Droid - Клиент свободных репозиториев приложений андроид.

    KeePassDX - Менеджер паролей для андроида.

    Book Reader - Лучшая читалка для андроиде поддерживающая конвертацию pdf под размер экрана, поддерживает fb2, html, txt, epub, mobi, rtf, doc, pdf, djvu.

    DriveDroid - Эмуляция загрузочной флешки с помощью смартфона.
